# EPCMTP-Online

[EggyPartyCopyMapTextProcessor](https://github.com/AloneVoyager/EggyPartyCopyMapTextProcessor) 的网页版本

## 简介

当你从《蛋仔派对》游戏中复制地图分享文本（例如：`复制打开【蛋仔派对】，游玩地图《地图名》，地图码1234-5678。`）时，本网页可以自动解析其中的**地图名**和**地图码**，并生成格式化的标题与简介内容，方便用于视频标题、帖子发布或存档记录。

## 功能

- 自动提取地图名（`《》` 中的内容）
- 自动提取地图码（支持 `1234-5678` 格式）
- 生成预设格式的输出内容

## 使用方法

1. 在游戏中复制地图分享文本（点击地图介绍界面的地图码复制）
2. 打开[本网页](https://alonevoyager.github.io/EPCMTP-Online/)
3. 根据网页提示，粘贴输入的文本并点击“处理文件”
4. 网页自动将文本格式化并显示输出结果

### 输入示例

```
复制打开【蛋仔派对】，游玩地图《梦幻游乐园》，地图码5821-3497。
```

### 输出内容示例

```
【蛋仔派对】地图《梦幻游乐园》试玩

相关游戏：蛋仔派对
地图名：梦幻游乐园
地图码：5821-3497
(EggyPartyCopyMapTextProcessor Generate)
```

## 错误处理

如果解析失败，网页会显示错误信息。

常见错误：
- 未找到地图名（缺少 `《》`）
- 未找到地图码（缺少地图码字段）
